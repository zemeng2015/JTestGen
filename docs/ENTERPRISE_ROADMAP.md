# Enterprise Roadmap

The open-source CLI is the foundation, but enterprise and team usage requires workflow, governance, and deployment controls around the generation loop.

## Team workflow features

- batch coverage remediation
- PR generation
- GitHub Actions workflow
- Jenkins integration
- HTML report dashboard
- run history

## Governance features

- team-specific test rules
- prompt/version tracking
- policy-based target selection
- generated test review checklist
- audit logs

## Private deployment features

- BYOK
- OpenAI-compatible private endpoints
- Azure OpenAI support
- local model endpoint support
- no production source modification mode
- artifact export

## Future integrations

- GitHub App
- GitLab merge request integration
- SonarQube / JaCoCo report ingestion
- Slack/Teams reporting
